# codeup-insight

**Codeup Insight** 是一款 ** 云效 Codeup 平台设计的代码贡献度分析工具**。
系统运行于浏览器端，通过分析云效 Codeup 的仓库提交记录，从**多角度**推断个人与团队的功能贡献情况，并结合大模型生成可读性强的功能画像结论。

> 🎯 目标：
> 不依赖后端，回答真实研发管理中的核心问题。

---

## ✨ 核心能力

-   分析每位成员主要负责的功能 / 模块
-   识别团队整体功能覆盖与协作情况
-   支持多仓库、多成员的可视化对比
-   基于 LLM 自动生成“功能画像总结”

## 🔐 存储策略与安全

### 存储策略

-   所有凭证仅存储在：
    -   `localStorage`
    -   浏览器内存

### 安全原则

-   Token 不持久化到任何服务器
-   所有 API 请求透明、可审计

---

## 🔗 API 调用方式

-   使用 `axios`
-   所有请求直接从浏览器发送至：
    -   阿里云 Codeup OpenAPI
    -   大模型（LLM）API Endpoint

## 📌 定位说明

**codeup-insight** 不是代码审查工具，也不是绩效系统。
它是一个**以事实数据为基础、以功能贡献为核心的工程洞察工具**。

### 适用人群

-   技术负责人
-   架构师
-   研发管理者
-   希望了解团队真实协作结构的工程团队
